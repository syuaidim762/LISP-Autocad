# 🚀 My AutoLISP Collection

Koleksi script buatan sendiri untuk efisiensi drafting di AutoCAD.

### 📋 Daftar Isi
| Nama Script | Perintah (Command) | Fungsi |
| :--- | :--- | :--- |
| **SumLines.lsp** | `TLEN` | Menghitung total panjang garis yang dipilih. |
| **AutoLayer.lsp** | `ALAY` | Memindahkan objek ke layer standar otomatis. |

### 📥 Cara Instalasi
1. Download file `.lsp` dari folder di atas.
2. Di AutoCAD, ketik `APPLOAD`.
3. Pilih file yang sudah di-download, klik **Load**.
4. Ketik perintah (Command) yang sesuai di command line.
